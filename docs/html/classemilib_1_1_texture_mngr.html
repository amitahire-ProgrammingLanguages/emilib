void emilib::TextureMngr::prepare_eviction ( )

When we need to load a bunch of new things it is prudent to throw out the old. We do this in three steps: 1) call prepare_eviction 2) call prefetch (or get) on all texture you are planning to use 3) call finalize_eviction, which will, in order:

  • throw out all texture not marked in step 2.
  • load all textures prefetched in step 2
