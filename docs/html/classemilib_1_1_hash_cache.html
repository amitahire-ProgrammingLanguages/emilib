Detailed Description

template<typename T>
class emilib::HashCache< T >

Wraps a value and memoizes the hash of that value. This can be used to speed up std::unoredred_set/unoredred_map or emilib::HashSet/HashMap when calculating the hash of the key is expensive. Example: emilib::HashMap<emilib::HashCache<Key>, Value>
